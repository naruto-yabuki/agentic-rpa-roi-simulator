# Agentic RPA 削減効果シミュレーター

株式会社ニューロスフィア「Agentic RPA」の一次商談で使う、コスト削減効果の試算ツール。営業担当が対象業務（受注処理・請求書発行など8種類）を選び、顧客に2〜3問ヒアリングするだけで、その場で削減額・ROI回収期間を算出して見せられる。

要件定義書: [`docs/要件定義書_料金計算Agent.md`](docs/要件定義書_料金計算Agent.md)（設計: Fable 5 / 実装: Sonnet 5）

## 画面構成

| 画面 | パス | 役割 | 顧客に見せるか |
|---|---|---|---|
| メイン画面 | `/` | 商談中のヒアリング入力＆試算結果表示 | 見せる |
| 設定ページ | `/settings` | 平均年収・業務別デフォルト値などの事前カスタマイズ（商談前に営業が調整） | 見せない |

## 計算ロジックの要点

- 人件費は「平均年収（既定600万円）÷ 年間所定労働時間（既定2,000時間）」で時給換算する（ハードコードしない）
- 業界ではなく「対象業務」（受注処理・請求書発行・経費精算など8種類）を選ぶことで1件あたり処理時間・自動化率のデフォルト値を自動補完するため、顧客への質問は実質2問（担当人数・処理件数/日）のまま試算できる
- 担当人数は工数の物理上限（キャップ）算出・専従率表示・人数換算表示に使う
- 設定ページの変更はメイン画面に即時反映される（React Context + localStorage、DB永続化なし）
- 選んだ業務単体では黒字化しない場合も「回収不可」と切り捨てず、あと何万円/月の削減で黒字化するかを示した上で、他業務との組み合わせを提案する

## セットアップ

```bash
npm install
npm run dev   # http://localhost:3400
```

## テスト

```bash
npm run test        # calc.ts のシナリオ1〜5をユニットテスト
npx tsc --noEmit    # 型チェック
npm run build        # 本番ビルド
```

## データの取り扱い

外部API・DB・サーバー送信は一切なし。入力値（顧客名含む）はブラウザ内のみで完結する。
